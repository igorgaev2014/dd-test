Введение в Javascript

Давайте посмотрим, что такого особенного в JavaScript, чего можно достичь с его помощью и какие другие технологии хорошо с ним работают.

Что такое JavaScript?

Программы на этом языке называются скриптами. Они могут встраиваться в HTML и выполняться автоматически при загрузке веб-страницы.

Что может JavaScript в браузере?

Современный JavaScript – это «безопасный» язык программирования. Он не предоставляет низкоуровневый доступ к памяти или процессору, потому что изначально был создан для браузеров, не требующих этого. В браузере JavaScript может:

  • Добавлять новый HTML-код на страницу, изменять существующее содержимое, модифицировать стили.
  • Реагировать на действия пользователя, щелчки мыши, перемещения указателя, нажатия клавиш.
  • Запоминать данные на стороне клиента («local storage»).

Что делает JavaScript особенным?

Как минимум, три сильные стороны JavaScript:

  1. Полная интеграция с HTML/CSS.
  2. Простые вещи делаются просто.
  3. Поддерживается всеми основными браузерами и включён по умолчанию.
